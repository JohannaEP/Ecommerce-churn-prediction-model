📊 E-commerce Churn Prediction Model
Modelo completo para analizar y predecir el churn (abandono de clientes) en plataformas de e‑commerce. Incluye EDA, modelado de machine learning, segmentación de clientes y visualización de métricas clave para apoyar decisiones de retención.

🧠 Objetivo del Proyecto
Identificar patrones de comportamiento que anticipen el abandono de clientes y construir un modelo predictivo que permita clasificar el riesgo de churn, facilitando estrategias de retención basadas en datos.

📂 Contenido del Proyecto
EDA (Análisis Exploratorio de Datos)  
Limpieza, tratamiento de valores faltantes, análisis de correlaciones y comportamiento del cliente.

Definición de Churn  
Criterio configurable de inactividad y generación de etiquetas churn / activo.

Modelado Predictivo  
Entrenamiento y comparación de modelos (Regresión Logística, Árboles, Random Forest, etc.) con métricas como accuracy, recall, F1 y AUC.

Segmentación de Clientes  
Clasificación en grupos de riesgo (alto, medio, bajo) y descripción de perfiles.

Dashboard Analítico  
Visualización de métricas clave, importancia de variables y distribución por segmentos.

Recomendaciones de Retención  
Insights accionables basados en patrones detectados.
